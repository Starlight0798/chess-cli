# Chess CLI
一个异步的跨平台(windows, macOS, Linux)的中国象棋命令行脚手架，致力于跨平台且在CMD中达到接近于GUI的体验，支持在CMD下启动多核象棋引擎，不再局限于使用Windows下的GUI来操作引擎。

# Features
- 支持引擎：
    - [皮卡鱼](https://www.pikafish.com/)
- 支持引擎协议
    - [UCI协议](https://www.xqbase.com/protocol/cchess_ucci.htm)
